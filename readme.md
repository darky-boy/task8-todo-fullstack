# To-Do List Application (Assignment 8)

This is a full-stack To-Do List application built using Node.js, Express.js, MongoDB, and React.

## Tech Stack
- Frontend: React, Axios
- Backend: Node.js, Express.js
- Database: MongoDB
- Deployment: Netlify (Frontend), Render (Backend)

## Features
- Create, read, update, and delete tasks
- Update task status
- API-based backend
- React frontend integrated with backend APIs

## Deployed Links
- Frontend: https://tutedude-todo-om.netlify.app
- Backend: https://task8-todo-backend.onrender.com
